# Lesson 2: JavaScript Data Types, Calculations, and the Browser Console 📊

Welcome to Day 2 of JavaScript! This lesson introduces how data is categorized, stored, and manipulated. We will explore the browser developer console to test snippets, view logged data, and debug errors.

## ⚠️ Getting Started (Important Instructions)
To work on these tasks, you must set up your own copy of the project:
1. Click the **"Fork"** button at the top right of this repository to create a copy under your own GitHub account.
2. **Clone** your personal forked repository to your local machine using your terminal:
   ```bash
   git clone <your-forked-repo-url>
   ```
3. Open the folder in **VS Code** and practice the tasks locally on your computer.
4. Right-click `index.html` and choose **Open with Live Server** or **Reveal in Finder or File Explorer**
5. **DO NOT create a Pull Request (PR)** back to the main classroom repository. All your work stays in your own personal repository!
6. At the end of class, you will save, commit, and push your working responsive design to your personal repository!

## 📝 What You Will Learn
* The core primitive data types in JavaScript
* How to open and navigate the browser developer console
* How to check data types using the typeof function
* How to perform basic mathematical equations and calculate remainders using modulo
* How to output clean messages from VS Code directly to your browser panel

## 🛠️ Classroom Tasks

1. **Explore Data Types:** Identify and create variables for the primary JavaScript data types:
   * String: Text wrapped in quotes (e.g., "Ghana").
   * Number: Whole or decimal numbers without quotes (e.g., 21, 19.99).
   * Boolean: Pure true or false logical states.
   * Null: An explicit, intentional empty value.
2. **The Developer Console:** Learn to right-click a web page, select Inspect, and navigate to the Console tab to:
   * Log live output data.
   * Spot and debug red syntax errors.
   * Test code lines on the fly.
   * Use the typeof function to print out variable classifications.
3. **Mathematical Calculations:** Perform operations directly in your code and log the results:
   * Addition (+)
   * Subtraction (-)
   * Multiplication (*)
   * Division (/)
   * Remainder using Modulo (%)
4. **Console Logging:** Master the console.log() function to send custom developer notes to the screen.
5. **String Interpolation Practice:** Use template literal backticks (``) to cleanly log dynamically combined text messages from your VS Code editor straight to the browser console.

## 🚀 How to Practice
1. Open your browser's Developer Console to run quick math equations like 50 * 2 or check types using typeof true.
2. Write your main calculation scripts inside your local script.js file.
3. Save your code, refresh your web browser tab, and inspect the console tab to read your outputs!
